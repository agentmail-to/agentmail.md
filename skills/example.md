# Example skill

This is a template. Each skill is a single self-contained markdown file with:

- **frontmatter** — `name` (kebab-case) and `description` (one line, used to
  decide when the skill is relevant).
- **body** — the instructions, examples, and API calls the agent should follow.

Write the body in plain markdown. Fenced code blocks, tables, and links all work.

```bash
curl https://agentmail.md/skills/example.md
```

Replace this file with your own curated skills.
